
# radiodb-viewer

source code of https://radiodb.freeddns.org

## Installation

1. Install dependencies:
   * make
   * docker
   * inotify-tools

2. Build and run:
   ```
   make dev
   ```

3. Visit:
   ```
   http://localhost:7446
   ```
